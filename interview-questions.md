# Bash Scripting Interview Questions

## What is Bash?

Bash is a command-line shell and scripting language used in Linux.

## Why do we use shell scripts?

Shell scripts help automate repetitive administrative tasks.

## How do you execute a shell script?

chmod +x script.sh
./script.sh

## What is the shebang line?

#!/bin/bash

It tells Linux to execute the script using Bash.

## What are positional parameters?

Positional parameters are values passed to a script as arguments.

## What is the purpose of automation?

Automation reduces manual effort, improves consistency, and saves time.
